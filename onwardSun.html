Website Redesign

Overview

This was my Application Project (INFO-5139) completed during the Web Development and Internet Applications program at Fanshawe College. I worked as part of a small team where we chose to redesign the website of a local business. I contributed as a frontend developer and also handled backend integration work using Firebase. The redesigned site is currently being prepared for deployment.
